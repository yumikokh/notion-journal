# Nikki — プロダクト仕様

## コンセプト

「Notionジャーナルをスマホで書きやすく、AIで週次に振り返りやすく」

## ターゲット

- Notionでジャーナルをつけていてモバイル入力に不満がある人
- 内省・セルフコーチングに興味があるフリーランス・クリエイター

## 画面構成

```
[タブバー]
├── 📝 Today（デイリー入力）
├── 📅 Calendar（カレンダー + 一覧）
├── 💬 Reflect（AI週次分析）
├── 📊 Insights（ダッシュボード）
└── ⚙️ Settings
```

### Today画面
- 上部: 日付 + Feelingピッカー（顔文字5種タップ選択）
- 中部: Good / Highlight / Struggle / Next の4枚カード（スワイプまたはタブ切替）
  - 各カードは箇条書き入力に最適化（改行で自動リスト化）
- 下部: 習慣チェックボックス横一列（Output, Book, Design, English, Exercise）
- フローティングボタン: 自由記述エディタ展開（Markdown対応フルスクリーン）

### Calendar画面
- 月間カレンダー（各日にFeeling顔文字 + 記録有無ドット）
- 日付タップ → エントリ詳細表示

### Reflect画面（AI週次分析）
- 週ピッカー（今週/先週/任意週）
- 「分析する」ボタン → AIがその週のジャーナル + カレンダー実績を読み、ワンショットで結果を出力
- 結果セクション: サマリー / 気付き・パターン / KPT案 / 次週フォーカス
- KPT案カードに「Notion KPT DBに保存」アクション
- チャットUIは持たない（ターン制の会話ではなく、週次ワンショット分析）

### Insights画面
- Feeling推移（週・月グラフ）
- 習慣達成率
- ジャーナルストリーク

## 競合との差別化

1. **Notion双方向同期** — 競合ゼロ
2. **構造化入力 × 自由記述** — プロパティクイック入力 + 本文記述の両立
3. **既存データ活用** — 初日からAIにデータがある
4. **日本語ネイティブ最適化**
