# Nikki — プロダクト仕様

## コンセプト

「Notionジャーナルをスマホで書きやすく、AIで週次に振り返りやすく」

## ターゲット

- Notionでジャーナルをつけていてモバイル入力に不満がある人
- 内省・セルフコーチングに興味があるフリーランス・クリエイター

## 画面構成

```
[タブバー]
├── 📝 Today（デイリー入力）
├── 📅 Calendar（カレンダー + 一覧）
├── 💬 Reflect（AI週次分析）
├── 📊 Insights（ダッシュボード）
└── ⚙️ Settings
```

### Today画面
- 上部: 日付 + Feelingピッカー（顔文字5種タップ選択。**その日の総合**気分）
- 中部: **Moment タイムライン** — その日に投稿した時刻つきライフログを時系列表示
  - SNS的に「今の気分・出来事」を1日に何度でも投稿 → Daily ページ本文に callout として append
  - 各 Moment は 時刻 + Feelingアイコン + 本文（#tag 可）
- 下部: 習慣チェックボックス横一列（Output, Book, Design, English, Exercise）
- フローティングボタン: **Moment 投稿**（本文 + その瞬間の Feeling）
  - 長文を書きたい時は同じ本文エディタで自由記述（Markdown）も可。充電/放電などの吐き出しはここに書く
- 構造化フィールド（Good/Highlight/Struggle/Next の4カード）は持たない。書きたいことは本文へ

### Calendar画面
- 月間カレンダー（各日にFeeling顔文字 + 記録有無ドット）
- 日付タップ → エントリ詳細（その日の Moment タイムライン + 自由記述 + 習慣/Feeling）
- 「気軽に過去を見返す」体験の中心。感情の記憶の振り返りはここと Insights が担う

### Reflect画面（AI週次分析）
- 週ピッカー（今週/先週/任意週）
- 「分析する」ボタン → AIがその週のジャーナル + カレンダー実績を読み、ワンショットで結果を出力
- 結果セクション: サマリー / 気付き・パターン / KPT案 / 次週フォーカス
- KPT案カードに「Notion KPT DBに保存」アクション
- チャットUIは持たない（ターン制の会話ではなく、週次ワンショット分析）

### Insights画面
- Feeling推移（週・月グラフ）
- 習慣達成率
- ジャーナルストリーク

## 競合との差別化

1. **Notion双方向同期** — 競合ゼロ
2. **構造化入力 × 自由記述** — プロパティクイック入力 + 本文記述の両立
3. **既存データ活用** — 初日からAIにデータがある
4. **日本語ネイティブ最適化**
